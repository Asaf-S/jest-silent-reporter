## Installation

Using npm:

```bash
$ npm i -D jest-silent-reporter
```

Using yarn:

```bash
$ yarn add -D jest-silent-reporter
```

## Usage

Jest CLI:

```bash
jest --reporters jest-silent-reporter
```

Jest Config:

```json
{
    "reporters": ["jest-silent-reporter"]
}
```
